# stick_figure_generator
<br>
Stick‑Figure Image Generator (create a selectable amount of test images)
<br><br>
<img width="196" height="183" alt="figure_gui" src="https://github.com/user-attachments/assets/c613a0b6-0ac6-450a-8866-6e81378b3357" />
<br><br>
Each image contains:
<br><br>
* a stick‑figure on the left (black background, red joints, limb segments
  in unique colours that are fixed across all images).
<br>
* the image number on the right (if enabled).
